# kitty 
- Split vertical: Ctrl + Shift + s
- Split horizontal: Ctrl + Shift + a
- Close active panel: Ctrl + Shift + w
- Resize active window: Ctrl + Shift + r

# nvim
- Search current file: / 
- Search word under cursor: * 

- Go to definition: gd 
- View definition in split window: gs 
- Close the split window: Space + q
- Jump back to location before gd: Ctrl + o 
- Go to references: gr 
- Go to implementation: gi

- Hover documentation: Shift + K
- Hover error detail: Shift + E

- Find file name: Space + /
- Fulltext search: Space + Space 
- Search buffer: Space + , 
- Global find & replace: Space + sR 
- Browse files from here: - 

- Undo: u 
- Redo: R 

- Jump from file to Neotree: Ctrl + w + h 
- Add new file in Neotree: a 
- Rename (Move) file in Neotree: m 
- Delete file from Neotree: r

- Git satus: gs 
- Git blame: gb 

- Leap forward: s 
- Leap backward: S 

# Windows Manager 
- Tile left: Super + Left 
- Tile Right: Super + Right 
- Tile Top: Super + Top 
- Tile Bottom: Super + Bottom 
- Maximize: Ctrl + M 
- Hide: Ctrl + H
- Close/Quit: Ctrl + W
